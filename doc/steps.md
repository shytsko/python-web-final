1. Создать новый каталог для проекта

```
mkdir python-web-final
cd .\python-web-final\
```

2. Инициализация git

`git init`

3. Создать и активировать виртуальное окружение

```
python -m venv .venv
.\.venv\Scripts\activate
```

4. Создать файл .gitignore
```
/.venv/
/.idea/
**/__pycache__/
*.py[cod]
/.env
```
